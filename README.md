# ai-story-generator
“A web application that generates personalized stories using an OpenAI model. Built with HTML, CSS, JavaScript, Bootstrap, Node.js and MongoDB. Includes AI text generation through the OpenAI API and a simple, user‑friendly interface.

## Assets & third‑party libraries (not included)

This repository does **not** include some images and third‑party libraries (e.g. Turn.js) because they may be subject to copyright / restrictive licensing.

To run the project locally:
1. Add your own images into:
   - `images/`
   - `imagescopy/`
2. Download Turn.js from the official website and place it here:
   - `libraries/turnjs4/`
3. Start the backend server
